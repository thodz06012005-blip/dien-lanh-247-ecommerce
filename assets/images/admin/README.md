# Ảnh dành cho trang quản trị

Chỉ đặt hình minh họa dùng riêng trong Admin Workspace tại đây, ví dụ onboarding, empty state hoặc hướng dẫn thao tác.

- Không sao chép ảnh sản phẩm vào thư mục này; Admin nên hiển thị cùng URL ảnh sản phẩm từ API.
- Tên: `admin-<screen>-<purpose>.svg` hoặc `.webp`.
- Không chứa thông tin khách hàng thật, token, email nội bộ hoặc ảnh chụp màn hình có dữ liệu nhạy cảm.
- Empty state ưu tiên SVG nhẹ dưới 60 KB.

Khai báo vị trí dùng trong `assets/images/manifest.json`.
