# Ảnh bài viết

Dùng cho `/articles`, trang chi tiết bài viết và Open Graph.

- Tên cover: `article-<article-slug>-cover.webp`.
- Tỷ lệ: 16:9, khuyến nghị 1600×900.
- Tối đa 160 KB.
- Ảnh trong nội dung: `article-<article-slug>-01.webp`, `-02.webp`.
- Không chèn tiêu đề dài trực tiếp vào ảnh; tiêu đề phải là HTML để SEO và responsive tốt hơn.

Xem mã ảnh và file seed liên quan trong `assets/images/manifest.json`.
